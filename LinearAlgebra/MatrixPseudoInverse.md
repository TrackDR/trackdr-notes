From https://inst.eecs.berkeley.edu/~ee127/sp21/livebook/def_pseudo_inv.html

$A$ is a $m$ x $n$ matrix

$$\begin{eqnarray}
A &amp;=&amp;
U
\left[ \begin{array}{cc}
S &amp; 0 \\
0 &amp; 0
\end{array} \right]&nbsp;V^T\\
A^\dagger &amp;=&amp;
V
\left[ \begin{array}{cc}
S^{-1} &amp; 0 \\
0 &amp; 0
\end{array} \right]&nbsp;U^T\\
\end{eqnarray} $$
